{% translate "Getting started" %}

{% translate "What is Terminklick?" %}

{% translate "Terminklick gives your business a public booking page. Customers pick a staff member, a service, and a time slot, and the booking lands directly in your dashboard. No phone calls needed." %}

{% translate "Do my customers need to create an account?" %}

{% translate "No. Customers book with just their name, email, and phone number. They manage or cancel their booking later through a private link sent by email, without ever registering." %}

{% translate "How long does setup take?" %}

{% translate "Most businesses add their staff, services, and availability, and publish their booking page within the same day. There's no onboarding call required." %}

{% translate "Is there a limit on staff, services, or bookings?" %}

{% translate "No. Your plan includes unlimited staff members, services, and bookings." %}

{% translate "Bookings and availability" %}

{% translate "Can I review bookings before they're confirmed?" %}

{% translate "Yes. Choose manual confirmation in your settings and every request waits for your approval. Or leave it on automatic and bookings confirm themselves the moment a slot is chosen." %}

{% translate "How do I set my availability?" %}

{% translate "Define recurring weekly hours in a few clicks, or add one-off blocks for specific dates. You can change either at any time." %}

{% translate "Can two customers book the same time slot by mistake?" %}

{% translate "No. Once a slot is booked or has a pending request, it's no longer offered to other customers, so double-booking isn't possible." %}

{% translate "Can a customer cancel their own booking?" %}

{% translate "Yes, at any time, using the private link they received by email. No account or phone call needed." %}

{% translate "Security and privacy" %}

{% translate "Is my account protected by two-factor authentication?" %}

{% translate "You can turn on optional two-factor authentication (TOTP, using any standard authenticator app) with single-use backup codes in your account settings." %}

{% translate "Does Terminklick track my customers?" %}

{% translate "No. There are no third-party trackers, ad cookies, or analytics scripts on the public booking page or your dashboard." %}

{% translate "What happens to customer data over time?" %}

{% translate "Customer contact details are anonymised automatically a set number of days after the appointment. Customers can also request an export or deletion of their own data at any time through their private booking link." %}
